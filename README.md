# sales-dashboard-analysis

## Objective
Analyze sales performance across categories, regions, and customer segments using Excel.

## Dataset
Superstore Sales Dataset

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design

## KPIs
- Total Sales
- Total Orders
- Categories
- Regions

## Key Insights
- Technology generated the highest sales.
- Sales increased significantly during Q4.
- Regional sales performance varied across markets.
